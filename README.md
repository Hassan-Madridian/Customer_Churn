# Customer Churn Prediction

## Business Problem
Predict which customers are likely to churn so the business can take retention actions.

## Dataset
Telco Customer Churn dataset (Kaggle).

## Approach
- Data cleaning & preprocessing
- EDA on churn patterns
- ML models (Logistic Regression, Random Forest)
- Evaluation using ROC-AUC, Precision, Recall, F1

## Results
Random Forest achieved the best ROC-AUC and identified key churn drivers.

## Business Insights
- Month-to-month contracts churn more
- Higher monthly charges correlate with churn
- Short tenure customers are at higher churn risk

## Tools
Python, Pandas, Scikit-learn, Matplotlib
